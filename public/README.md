# Character Images

Place your custom character images in this folder:

- `ncp.png` - NCP character image (recommended size: 64x64px or 128x128px)
- `bnp.png` - BNP character image (recommended size: 64x64px or 128x128px)
- `jamat.png` - JAMAT character image (recommended size: 64x64px or 128x128px)
- `bal-enemy.png` - BAL enemy image (recommended size: 64x64px or 128x128px)

The game will automatically load these images if they exist, otherwise it will use the default geometric shapes.

**Image Requirements:**
- Format: PNG with transparent background recommended
- Size: 64x64px to 128x128px works best
- Square images work better for the game

**Example file structure:**
```
public/
├── ncp.png
├── bnp.png
├── jamat.png
└── bal-enemy.png
```
